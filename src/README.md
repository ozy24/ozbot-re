# Source Code Directory

This directory contains all the source code files for the Quake 2 mod template.

## Overview

All `.c` and `.h` files in this directory are automatically compiled by `build.bat` in the parent directory.

## File Organization

- **Core game files** (`g_*.c`) - Main game logic
- **Player files** (`p_*.c`) - Player-specific code
- **Monster files** (`m_*.c`) - Monster AI and behavior
- **Header files** (`*.h`) - Definitions and structures
- **Shared code** (`q_shared.c/h`) - Common utilities

## Adding New Files

Simply add new `.c` files to this directory and they will be automatically included in the build.
